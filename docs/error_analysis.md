# Error Analysis

## Observations
- Some sound classes are acoustically similar
- Noise affects low-energy signals
- Band-limiting removes high-frequency features

## Confusion Matrix Insights

The confusion matrix shows that certain classes are frequently misclassified due to similar spectral characteristics. This is expected for environmental sounds with overlapping frequency content. The model performs better on distinct sound classes with unique spectral signatures.

## Challenges
- Overlapping spectral characteristics
- Reduced distinguishability under distortions

## Conclusion
Despite improvements, some classes remain difficult due to inherent similarity.
